## História
O camaleão Pascal precisa percorrer a floresta do Reino de Corona coletando lanternas mágicas. Durante a jornada, o jogador deve desviar de obstáculos como pedras, arbustos e troncos para sobreviver o maior tempo possível. Além das lanternas, ocasionalmente surge a Flor Mágica, um item raro que concede uma quantidade maior de pontos. O objetivo é obter a maior pontuação possível e registrar seu nome no ranking de melhores jogadores.
